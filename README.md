# AI 泡沫流動性檢測（AI Bubble Liquidity Check)

本 repo 為**研究筆記與設計文件庫**（非程式碼專案），目標是為一個「月度流動性儀表板」建立可直接實作的藍圖：透過三層流動性（政策／市場信用／投機）加上 AI 資本支出對 EPS 的傳導橋，監測美股 AI 泡沫風險。儀表板尚未實作，目前處於規格（spec）階段。

```

```
